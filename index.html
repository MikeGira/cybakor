Image mode for NexOS Enterprise Linux

Contain drift.
Accelerate delivery.

Your systems should be as easy to update as smartphones.

Image mode provides a simple, consistent approach to build, deploy, and manage the OS using container technologies. It reduces drift by keeping your servers' configurations the same—eliminating deviations that lead to system instability and increased security risks.

See the details
✦
bash — Image Mode Build & Deploy
$ cat Containerfile FROM nexos/nexos:10 RUN dnf install -y nginx python3.12 postgresql15 COPY hardening/ /etc/nexos-sec/ RUN nexos-image seal --fips --signed   $ nexos-deploy push --target=prod-fleet Pushing OCI image to registry… [████████████████████] 100% Rolling out to 300 nodes in parallel… ✓ Done in 4m 12s. Zero drift detected. ✓ All nodes identical. No reboot required.
